## Introduction

~~to make some...~~

莫名其妙从麦趣鸡盒联想到了游戏盒，就感觉很有意思于是有了这个游戏盒。

计划为这个游戏盒添加一些诸如魔塔、flappy board之类的经典小游戏。

既然是麦趣鸡盒，会原生去实现体积大概会小一点，大概。

## install(for dev)

1. **download or clone this project**

2. **enter the root directory**

3. **run shell**

   * ```bash
     npm i
     ```

     **or**

   * ```bash
     yarn
     ```
